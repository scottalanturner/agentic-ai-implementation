# Submitting Assignments via GitHub

## Getting Started

1. Sign up for a free account at [github.com](https://github.com) if you don't already have one.
2. Create a new repository for the course (e.g., `isys398u-assignments`). Make it public so it's visible to the world.

## Organizing Your Work

- Put each assignment in its own folder named after the assignment (e.g., `Assignment01`, `Assignment02`).
- Include a `README.md` file inside each folder with a brief description of what you did and any notes about your work.

## Submitting

Once your assignment is pushed to GitHub, copy the link to the relevant folder and paste it into the Blackboard submission box. Do not attach files directly to Blackboard.
