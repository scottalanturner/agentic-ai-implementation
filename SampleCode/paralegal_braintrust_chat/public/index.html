/capture — Start your message with /capture (then a space) to save this turn to the Braintrust dataset live-captures (or the name in BRAINTRUST_CAPTURE_DATASET in .env). The assistant does not see the word /capture; only your question is sent to the model. Use it in class to promote a good example into a dataset for later evals. If capture fails, check the red error line below or the server console.
